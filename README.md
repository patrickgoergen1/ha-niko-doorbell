# Niko Doorbell

A Home Assistant custom integration for the Niko Doorbell (NHC1).

## Features

- RTSP Camera
- Call Status Sensor
- Mute Status Sensor
- Mute Switch
- Hangup Button
- Config Flow
- HACS compatible

## Supported devices

Currently tested:

- Niko Doorbell 550-22001
